configファイルの書き方
背景
	対象のディレクトリに入っている.exeと.rbを実行
	ただし、一部のファイルは無視したい
１．
	名前_config.txtという名前のファイルを作る
	名前は何でもいい
	以下これをconfigファイルと呼ぶ
２．
	configファイルを次のように書く

	１行目：
		対象のディレクトリ（ディレクトリA）への相対パス

	２行目：
		pickup もしくは ignore
		３行目以降のものだけを拾うならpickup
		無視するならignore

	３行目以降：
		対象ディレクトリ・ファイルのディレクトリAからの相対パスのリスト

３．
	例
app/rei
ignore
test
sample

４．置くところ
	このreadmeとおなじところ
